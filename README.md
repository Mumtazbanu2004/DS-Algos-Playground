# DS-Algos-Playground
Small, focused Python examples for learning core programming and algorithm concepts.
Topics included
- Basics: functions, parameter passing, recursion
- Structured types: lists, classes, strings, file I/O
- Linear DS: stacks, queues, linked lists
- Trees: binary trees, BST insert/search/delete, traversals
- Heaps and heap sort
- Graphs: adjacency list, BFS, DFS, Dijkstra, Bellman-Ford, Kruskal, Prim
- Sorting and searching: merge sort, quicksort, binary search
- Algorithm design: greedy, dynamic programming, divide-and-conquer
- Utilities: simple hash table, complexity demos, timing scripts

Run examples
- Each topic is a single .py file. Run with `python3 filename.py`.
Contributions welcome. Open issues or PRs to improve examples or add tests.
